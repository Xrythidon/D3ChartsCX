## D3ChartsCX

An app that uses D3.js to create an beautiful and engaging charts.

## Tech/framework used

<b>Built with</b>

- React
- D3.js
- SCSS

## Features

- Insync Charting with integrable Data
- Mobile Design First
- SCSS Breakpoints
